# 🥗 Meal Planner App

An intuitive mobile application designed to help users effortlessly plan, organize, and manage their meals throughout the week.

> 🎨 Inspired by our Figma prototype: [Click here to view design](https://www.figma.com/design/2MQCMxiy1AOMOONfhSSvaS/meal-planner-app?node-id=0-1&p=f)

---

## 📌 Features

- 📅 Weekly meal planning interface
- 🛒 Visual layout for food categories
- 🍲 Add and manage meals visually
- 🔐 Designed with authentication flow (UI only)
- 🖼️ Based on modern, mobile-first UI principles

---

## 🛠️ Tech Stack

_This app was collaboratively developed for academic learning purposes._

- **UI/UX Design:** Figma
- **Front-End:** Likely built using React Native or Expo (based on folder structure)
